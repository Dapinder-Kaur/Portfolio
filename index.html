Deep Learning for Wildlife Conservation

Developed a deep learning solution to automate the detection and classification of wildlife species in remote sensing imagery, aimed at supporting global conservation efforts to protect endangered animals. Successfully built the model from a baseline CNN to an optimized architecture using MobileNetV2 transfer learning, achieving a final test accuracy of 93.67%. Augmented data (rotations, zooms, flips) and Dropout layers to generalize the model, ensuring high performance on images with complex backgrounds. Engineered a data-loading pipeline to parse and convert YOLO-format labels into categorical variables for training. Built a user-friendly web application allowing researchers to upload images and receive instant species identification with high confidence scores.

Python | TensorFlow | Keras | MobileNetV2 | Streamlit | OpenCV | NumPy | Scikit-learn
